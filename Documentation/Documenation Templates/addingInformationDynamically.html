Adding Information Dynamically

This guide will explain how to dynamically add information to any HTML project using the powerful jquery platform.

This guide assumes you already have jquery added to your HTML project. See this guide for more information on adding jquery to your project.

Step 1. Create a function.

 
   

function getListItems () {
	
}

Step 2. The first thing you need to add to your function is a JSON request which will pull the information from a JSON source. In this example we will be using a PHP Script as our source.

 
  
function getListItems () {
	$.getJSON('http://api.universalbusinessmodel.com/example_php_script.php?callback=?', {
		// This area will contain any variables that need to be sent to the script, in this example we are not 
		//passing any variables so this area is empty.
	}
}

Step 3. Now we need to add a function that gets the response (res) and the status from the PHP script.

 
 
function getListItems () {
	$.getJSON('http://api.universalbusinessmodel.com/example_php_script.php?callback=?', {
		
	}, function(res, status) {

	});	
}

Step 4. Next we need to take everything that was returned and append it to the unordered list that we will create later.

 
   
function getListItems () {
	$.getJSON('http://api.universalbusinessmodel.com/example_php_script.php?callback=?', {
		
	}, function(res, status) {
			$('#example_unordered_list').empty(); //Empties the list everytime we press our button, you can 
			//get rid of this line to see what happens without it.
		$.each(res, function(i, item) {
			$('#example_unordered_list').append("<li><p>" + item.list_item + "</p></li>"); // This will 
			//append the list item to the unordered list
			$('#example_unordered_list').listview("refresh"); // This will force jquery to refresh its CSS, you 
			//can get rid of this line to see what happens without it.
		})
	});	
}

Step 5. Now that we finished our JavaScript we can start on our HTML. The first thing we need to do is build the unordered list that our script will append to. 

   
<ul id="example_unordered_list" data-role="listview">
    
</ul>

Step 6. The last thing we need to do is create a button that will call our function

 


<button class="ui-btn">Append</button>

That's it! Below is everything put together for you to try.


<html>
	<head>
		<link rel="stylesheet" href="http://code.jquery.com/mobile/1.4.2/jquery.mobile-1.4.2.min.css" />
		<script src="http://code.jquery.com/jquery-1.9.1.min.js"></script>
		<script src="http://code.jquery.com/mobile/1.4.2/jquery.mobile-1.4.2.min.js"></script>
	<script>
	function emptyListItems () {
		$('#example_unordered_list').empty();
	}
	function getListItems () {
		$.getJSON('http://api.universalbusinessmodel.com/example_php_script.php?callback=?', {

			}, function(res, status) {
				$('#example_unordered_list').empty();
			$.each(res, function(i, item) {
				$('#example_unordered_list').append("<li>" + item.list_item + "</li>");
				$('#example_unordered_list').listview("refresh");
			})	
		});
	}
	</script>
	</head>
	<body>
		<ul id="example_unordered_list" data-role="listview">
		</ul>
		<div align="center">
				<button class="ui-btn" onclick="getListItems()">Append</button>
				<button class="ui-btn" onclick="emptyListItems()">Remove</button>
		</div> 
	</body>
</html>
